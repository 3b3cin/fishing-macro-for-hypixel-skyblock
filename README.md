# fishing-macro-for-hypixel-skyblock
a decently simple fishing macro that i made for the game hypixel skyblock

**how to set it up**
first download [minescript](https://minescript.net/)
after that place the file into your exec folder and simply type ingame \fishing_macro



i had a lot of fun making this project along with the minescript discord comunity
